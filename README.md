## Jalan-Jalan
A frontend project for recsys-ontology,
ontology-based recommender system.

### Features
- State management with Ngrx inspired by Redux
- Handling http request with Rxjs Observable

### Getting Started
- Clone this repo
- `npm install`
- `ionic serve`
